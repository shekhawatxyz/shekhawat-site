# shekhawat.site

Source for my personal website, built with [Hugo](https://gohugo.io) using the [Poison](https://github.com/lukeorth/poison) theme.

## Structure

- `content/projects/` — writeups of things I've built
- `content/updates/` — short daily/periodic logs of what I'm working on
- `content/about/` — about page
- `themes/poison/` — the Hugo theme (git submodule)

## Running locally

```sh
hugo server
```

## Building

```sh
hugo --minify
```
